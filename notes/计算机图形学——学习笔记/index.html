# 前言

终于迈出学习计算机图形学的第一步了......

喜欢上计算机图形学的起因大概是听了传奇程序员约翰・卡马克的故事，这位推动了 3D 发展的大佬给我留下了无比深刻的印象。

再加上我对渲染、光追、VR、AR 等等也都很感兴趣，就想去了解更多，于是也就有了这篇记录博客。
