# 编译 DOOM3

ID SoftWare 公司的经典作品，DOOM3

其实是因为我是约翰・卡马克的粉丝

直接选择编译 DOOM3-BFG 了，恰好 Github 上有大佬专门写了这部分的引擎，RBDOOM-3-BFG，就直接拿来编译了

然后就吃了不同发行版之间的亏......

ArchLinux 要安装以下的依赖

但是因为 SDL2 官方有两种打包方式，显然 ArchLinux 采用了另一种让，所以我在编译时一直找不到 SDL......

因此修改以下 CMakeLists.txt

编译完成后，将游戏的资源文件 base 放入 build 文件夹

注意，运行游戏时，会在～/.local/share/rbdoom3bfg 下生成游戏文件

如果重新编译使用记得清除旧游戏生成文件

vulkan 支持需要 git clone ----recursive
